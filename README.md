## 🚀 Деплой на GitHub Pages

1. Создай репозиторий на GitHub (например: `papa-birthday`)
2. Загрузи все файлы в корень репо
3. **Settings → Pages → Source** → ветка `main`
4. Готово! Через 1–2 мин: `https://ник.github.io/papa-birthday`
